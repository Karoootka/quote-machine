# quote-machine

Zadanie wykonane w ramamch bootcampu Kodilla - "Web developer"

Opis: Przećwiczenie wykorzystania technologii AJAX. Efektem jest aplikacja, która służy do pobierania i wyświetlania losowych cytatów z wykorzystaniem API (http://quotesondesign.com). Dodana została również możliwość udostępnienia danego cytatu na Twitterze.

Rozwiązania, technologie i narzędzia:

- HTML 5
- CSS 3
- RWD
- JavaScript
- AJAX
- API
